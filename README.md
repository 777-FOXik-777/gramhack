# что такое gramhack?
gramhack - это инструмент который поможет вам увеличить количество подписчиков в Instagram.
# Установка gramhack на Termux

+ apt update

+ apt upgrade -y

+ pkg install git

+ git clone https://github.com/777-FOXik-777/gramhack

+ cd gramhack

+ bash gram.sh
