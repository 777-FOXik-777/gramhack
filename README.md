# что такое gramhack?
installer - утилита в которой находятся самые крутые
Инструменты в термукс, с полностью автоматизированной установкой.

# Установка Installer на Termux

+ apt update

+ apt upgrade -y

+ pkg install git -y

+ git clone https://github.com/777-FOXik-777/gramhack

+ cd gramhack

+ bash gram.sh
